# Tugas Struktur Data - Web Apps dengan Streamlit

Proyek ini berisi dua aplikasi interaktif yang dibangun menggunakan Python dan framework Streamlit untuk memenuhi tugas mata kuliah Struktur Data.

## 🚀 Fitur Utama
1. **Set Operations Visualizer (`tugas1_set.py`)**
   - Visualisasi operasi himpunan (Union, Intersection, Difference, Symmetric Difference).
   - Input dinamis untuk elemen-elemen set.
2. **Social Media Word Count (`word_count.py`)**
   - Menganalisis frekuensi kemunculan kata dalam teks/komentar.
   - Membersihkan simbol dan tanda baca secara otomatis.
   - Menampilkan hasil dalam bentuk tabel/grafik sederhana.
